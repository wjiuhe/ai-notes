# Claude Code Skills for Mind & Machine

This directory contains reusable Claude Code skills extracted from the Mind & Machine website codebase.

## Skills Overview

| Skill | Description | Use Case |
|-------|-------------|----------|
| `dual-theme-web-system` | Parallel light/dark themes with CSS variables | Websites with multiple visual themes |
| `hand-drawn-css-components` | Sketchy, organic UI using pure CSS | Notebook/doodle-style interfaces |
| `sidebar-toc-navigation` | Fixed sidebar with scroll spy | Documentation with section navigation |
| `filter-sort-card-grid` | Filterable/sortable card grid | Index pages with filtering |

## Usage

These skills can be used with Claude Code:

```bash
# List available skills
claude skills list

# Use a specific skill
claude skills use dual-theme-web-system
```

## Compatibility

These skills follow the standard Claude Code skill format and are compatible with:
- Claude Code CLI
- Claude Desktop (with skill support)

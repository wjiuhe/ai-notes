Why Exactly 8? Hardware Limits!

Think of it like a whiteboard
The 37GB is like a whiteboard — you need space to work, but it clears after each patient! It's temporary activation memory needed per GPU during a forward pass, NOT the stored model weights.
More experts = more communication between GPUs = more waiting time!

Here's how the percentages are calculated:
  • 8 experts: Communication time fits perfectly inside computation time = 100% speed (no waiting!)
  • 10 experts: Communication takes 25% longer than computation = 25% slowdown
  • 12 experts: Communication takes 50% longer than computation = 50% slowdown
